# 🚀 Interactive Full-Stack Portfolio Experience

A modern, high-performance, and visually immersive portfolio application. This project showcases a seamless integration between a robust **ASP.NET Core** backend and a dynamic **React** frontend featuring **3D interactive elements**.

## 🌟 Key Features

* **Interactive 3D Experience:** Integrated **Three.js** via **React Three Fiber (R3F)** to render high-quality 3D models (like the "Falling Spaceman") for an engaging UX.
* **Global State Management:** Orchestrated complex data flows using **Redux Toolkit (RTK)** for predictable and scalable state handling.
* **Full-Stack Architecture:** A decoupled architecture with a **RESTful API** built on **ASP.NET Core 8.0** and a **Vite-powered React** frontend.
* **Secure Authentication:** Implemented **JWT (JSON Web Tokens)** and **Identity Server** logic for secure user/admin access.
* **Dynamic Localization:** Multi-language support (**TR/EN**) managed via **i18next**, allowing for a global audience.
* **Modern UI/UX:** Styled with **Tailwind CSS** for a "mobile-first" responsive design and **Framer Motion** for smooth transitions.

---

## 🛠️ Tech Stack

### **Frontend**
* **Framework:** React 18 (Vite)
* **State Management:** Redux Toolkit
* **3D Engine:** Three.js & React Three Fiber
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Localization:** react-i18next

### **Backend**
* **Framework:** ASP.NET Core 8.0 (Web API)
* **Database:** MongoDB (NoSQL)
* **Architecture:** Repository Pattern & DTOs
* **Authentication:** JWT Bearer Authentication
* **Mapping:** AutoMapper

### **Deployment & DevOps**
* **Hosting:** IIS (Windows Server) via Plesk
* **Configuration:** Custom MIME Type management for 3D asset delivery (.glb)
* **CORS:** Configured for secure cross-origin resource sharing
